AZURE RUNBOOK
AUTOMAÇÃO START/STOP de VM

Azure Runbooks são scripts automatizados criados e executados no Azure Automation, um serviço que permite gerenciar e automatizar tarefas e processos em sua infraestrutura de TI. Runbooks são fundamentais para realizar tarefas repetitivas, simplificar operações e implementar processos de gerenciamento contínuo.

Automação Inteligente - O escopo utilizando as identidades gerenciadas, contemplam os runbooks que são automações capazes de iniciar/parar máquinas virtuais em horários agendados definidos pelo usuário.

O que são Identidades Gerenciadas
Uma identidade gerenciada do Azure AD (Azure Active Directory) permite que o runbook acesse facilmente os outros recursos protegidos do Azure AD.
